# utils

JavaScript object &amp; array util methods.

## Object

- 判断数组或者对象是否为空
- 解析 query(string) 为 Object

## Array

- 创建二维数组
- 取数组里的某一属性组成新的数组
- 寻找数组中是否包含目标元素（兼容嵌套数组）

## String

- 替换字符串中的所有子串

## Date

- 将 utc 时间转换为可显示的本地时间：2019-01-28T08:27:19+00:00 -> 2019-1-28 16:27:19
